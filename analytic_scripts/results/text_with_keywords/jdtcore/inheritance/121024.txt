BUG REPORT COMMENTS:
In the following code, the call to createParser()
is reported as ambiguous by eclipse and not by javac.

public class AmbiguousEclipseBug {
    interface Listener {
    }
    interface ErrorListener {
    }  
    static Object createParser(Listener l) {
      return null;
    }
    
    static <L extends ErrorListener & Listener> Object createParser(L l) {
      return null;
    }
    
    public static void main(String[] args) {
      class A implements Listener, ErrorListener {
      }
      
      createParser(new A()); // error here
    }
}

Error reported :
The method createParser(AmbiguousEclipseBug.Listener) is ambiguous for the type AmbiguousEclipseBug.

This code seems legal, and <E>createParser() must be called
because this method is more specific than createParser().
