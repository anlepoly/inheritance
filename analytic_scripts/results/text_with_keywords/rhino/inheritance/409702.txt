BUG REPORT COMMENTS:
This is a regression sometime since 1.6R5.

Here is at least one that fails, running with JDK 1.6 in interpreted mode.  (Results are similar with JDKs 1.4-1.5, but the error messages are not as good.  Same with running with different values for optimization.)

Java:
package override;

public abstract class WithFinal {
	public WithFinal() {
	}
	
	public abstract void a();
	
	public abstract void b();
	
	public static abstract class Subclass extends WithFinal {
		public final void a() {
		}
	}
}

Rhino shell:
var instance = new JavaAdapter(
	Packages.override.WithFinal.Subclass,
	{
		b: function() {
		}
	}
);
print(instance);
